# Goldenmarkdelivery
## نظام التوصيل والتنسيق الميداني

Progressive Web App (PWA) for field coordination and delivery management.

### Deploy
Upload all files to GitHub Pages or Netlify.

### Files
- `index.html` — Main application
- `manifest.json` — PWA manifest
- `sw.js` — Service Worker (offline support)
- `icon-*.png` — App icons (all sizes)
